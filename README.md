# Vanshavali — Releases

Signed release APKs for the Vanshavali private family-tree app. **No source code is published
here on purpose** — this repo exists only to give family members a plain HTTPS link to download
app updates from, without needing GitHub access. The app's source lives in a separate, private
repository, restricted to vetted collaborators.

## Which APK do I install?

- **`vanshavali-admin-*.apk`** — for the person who owns/runs the tree (invites people, approves
  requests, ships updates). Only ever install this on the admin's own device.
- **`vanshavali-guest-*.apk`** — for everyone else joining the tree via an invite code.

Grab the latest from [Releases](../../releases). Both flavors are signed with the same release key
across versions, so installing a new version always updates in place over the old one — your tree
data, password, and access stay exactly as they were.
